动态规划

    本质 ： 递归问题或者是分治问题
    共性 ： 找出重复子问题
    差异性 ： 最优子结构，中途可以淘汰次优解
    优点 ： 时间复杂度降低
    时间复杂度=每个结点的时间复杂度*结点个数=O(1)*2^n=O(2^n)
    
    斐波那契数列递归写法的时间复杂度：
    T(0) = T(1) = 1
    T(n) = T(n - 1) + T(n - 2) + 2
    又因为
    F(0) = F(1) = 1
    F(n) = F(n - 1) + F(n - 2)
    所以根据数学归纳法，F(n) <= T(n)。
    又可以根据数学归纳法证明 n > 4 时，F(n) >= (3/2)^n，
    所以 n > 4 时，T(n) >= (3/2)^n。
    由此可以证明斐波那契数列递归算法至少具有指数时间的界。
